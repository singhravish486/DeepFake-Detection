About This Project

Learn about the technology and methodology behind our aerial deepfake detection system

Project Overview

This system detects manipulated (fake) aerial and satellite images using advanced artificial intelligence with explainable decision-making capabilities.

In an era where deepfake technology can create convincing fake imagery, it's crucial to have reliable tools for detecting manipulated aerial and satellite images. Our system addresses this challenge using state-of-the-art deep learning techniques combined with explainable AI.

Technical Architecture

CNN Branch (EfficientNet)

Purpose: Analyzes local features and small-scale artifacts

  • Detects texture inconsistencies
  • Identifies edge artifacts
  • Finds compression anomalies
  • Spots local manipulations

Vision Transformer (ViT)

Purpose: Captures global context and large-scale patterns

  • Global scene understanding
  • Long-range dependencies
  • Terrain consistency analysis
  • Large-scale manipulations

Explainable AI with Grad-CAM

Our system doesn't just tell you if an image is fake - it shows you exactly which parts of the image influenced that decision. Using Gradient-weighted Class Activation Mapping (Grad-CAM), we generate visual explanations that highlight the regions the AI model focused on.

Why Explainability Matters:
  • Trust: See the AI's reasoning
  • Legal: Evidence for court cases
  • Learning: Understand manipulation techniques
  • Debugging: Identify model biases
How Grad-CAM Works:
  1. AI processes the image
  2. Gradients flow back to feature maps
  3. Important regions are highlighted
  4. Visual heatmap is generated

Model Specifications

Architecture: Hybrid CNN + Vision Transformer
CNN Backbone: EfficientNetB0
Parameters: ~4.9M total (1.7M trainable)
Input Size: 224×224×3
Framework: TensorFlow 2.x
Training: Transfer Learning + Fine-tuning

Performance Metrics

Current Accuracy*: 50% (limited by dataset size)
Expected with full data: 85-95% accuracy
Analysis Time: < 1 second per image
Supported Formats: JPG, PNG, GIF, BMP, TIFF
Max File Size: 16 MB
Explainability: Grad-CAM visualizations
*Current model trained on limited dataset for demonstration

Current Limitations

  • Limited training dataset (100 images)
  • May have false positives on complex imagery
  • Optimized for specific types of aerial images
  • Requires further validation on diverse datasets

Future Improvements

  • Larger, more diverse training dataset
  • Multi-scale analysis capabilities
  • Real-time video processing
  • Integration with forensic analysis tools
